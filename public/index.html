API Hari Libur & Cuti Nasional
Indonesia

Solusi modern berbasis Bun untuk mendapatkan data hari libur resmi dan cuti bersama. Diperbarui otomatis setiap minggu melalui otomatisasi scraping.

Coba API Sekarang Baca Dokumentasi

Kinerja Kilat Bun

Dibangun di atas runtime Bun yang super cepat. Waktu respons instan dalam hitungan milidetik tanpa overhead berat.

Pembaruan Otomatis

Terintegrasi dengan GitHub Actions mingguan untuk melakukan scraping data terbaru dari `kalenderku.id` secara presisi.

Presisi Waktu Lokal

Menggunakan acuan waktu lokal Indonesia (GMT+8) sehingga pergantian hari di endpoint /today dan /tomorrow selalu akurat.

Dokumentasi Endpoints

Gunakan endpoint di bawah ini untuk mengakses layanan API publik.

GET /api/today
Cek libur hari ini

Mengembalikan informasi terstruktur apakah hari ini merupakan hari libur/cuti nasional beserta nama dan jenisnya.

GET /api/tomorrow
Cek libur besok

Mengecek status hari libur untuk keesokan harinya secara presisi menggunakan batas tanggal zona waktu lokal.

GET /api/this-month
Daftar libur bulan ini

Mengembalikan array seluruh tanggal libur dan cuti bersama yang jatuh pada bulan berjalan.

GET /api
Mendukung Filter Parameters

Mengambil daftar libur berdasarkan bulan dan/atau tahun tertentu.

?month={int} Menyaring data libur pada bulan ke-X (1-12)
?year={int} Menyaring data libur pada tahun ke-X (Contoh: 2026)
